  Important Issues

  1. 模組層級副作用（Important）— line 14, 110–131

  wb = openpyxl.load_workbook('interview_employee_data.xlsx')  # 模組載入即執行
  所有讀取 Excel、迭代資料列、呼叫 API 的邏輯都在模組頂層執行，import
  時就會觸發，導致無法測試、無法重用。建議包成 main() 函式。

  2. 廢棄的第一次迭代（Important）— line 18–19

  for row in ws.iter_rows(min_row=2, values_only=True):
      print(row)
  這個迴圈在正式處理之前多跑一次，只是 print，應移除。

  3. 未賦值的 f-string（Suggestion）— line 33

  f"""For each row..."""
  這是一個求值後被丟棄的 f-string，不是 docstring 也不是 comment，建議改為一般 #
  註解或直接刪除。

  4. validate_join_date 無法處理 None（Important）— line 82

  str(None) 會得到 "None"，會觸發 ValueError 並返回「invalid date
  format」，而非「join_date is required」，語意不清。

  5. validate_salary 允許 float（Suggestion）— line 76

  題目要求「正整數」，但目前 isinstance(value, (int, float)) 允許 50000.5 通過。
